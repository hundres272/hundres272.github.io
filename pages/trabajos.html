Mis trabajos

Simsomp API

Es una página interactiva que muestra los personajes de los Simpson, para esto la aplicación consume un API que permite acceder a los datos sobre cada uno de los personajes (nombre, imagen, frase). Esto con el fin de tener por separado la interfaz de usuario de los datos, permitiendo escalabilidad de la página a N cantidad de personajes. Esta página cuenta con responsive design que permite adaptarse a cualquier pantalla.

Components tienda virtual

Esta aplicación web simula una tienda online, dentro del diseño cuenta con productos que se cargan de forma dinámica desde la base de datos, detalles sobre los productos en existencia, carrito de compras para modificar y confirmar los detalles de la compra, inicio de sesión y creación de un usuario para las compras, entre otras muchas funciones de validación y presentación.

Contador cantidad de billetes

Es una aplicación java, fue pensada para ingresar valores numéricos que se validan en tiempo real y permiten hacer una transacción, los valores aceptados deben estar entre $10.000 y $500.000, debe ser múltiplo de $5.000 y entregará como resultado la cantidad de billetes de acuerdo a las denominaciones mostradas.
